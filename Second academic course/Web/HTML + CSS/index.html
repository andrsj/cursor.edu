Перша програма на C#

На даному уроці ми напишемо програму рішення квадратного рівняння!
Для перегляду теорії розгорни лівий квадрат, для рішення - правий!

Для того, щоб почати писати програму, треба спочатку написати алгоритм програми (або найти його) , щоб потім по цьому алгоритму писати код програми, яку хочемо реалізувати. Є два види подачі алгоритму, якими користуються найчастіше:

  • Текстовий
  • Візуальний

Текстовий алгоритм подається у вигляді рядку тексту, де описується порядок роботи програми, інколи використовують псевдокод (але можна тільки словесно описати)

  1. Ввести a,b,c
  2. Знайти дискримінант
  3. D > 0?
    • Так:
      • Знайти корені рівняння
      • Вивести їх значення
    • Ні:
      • D = 0?
        • Так:
        • Знаходимо значення кореня
        • Виводимо значення
        • Ні:
        • Знаходимо комплексі корені
        • Виводимо значення
  5. Кінець програми

Візуальний алгоритм подається у вигляді блок схеми, де кожна дія зображена в окремих квадратах, кружечках, прямокутниках і т.д. (Кожна фігура відповідає за окрему дію і має свої властивості)
