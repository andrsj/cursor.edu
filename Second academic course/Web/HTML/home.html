Перша програма!

Зараз ми ознайомимося як створити свою першу програму на С#!

Для початку треба ознайомитися з нашим компілятором для С#:
Зайдемо в консоль: Win + R -> cmd ; Tа введемо "csc.exe /? > help.txt". Після цієї команди ми можемо в файлі help.txt ознайомитися з командами компілятора csc.exe. Після ознайомлення з компілятором можна починати створення власної першої програми!
Створимо текстовий файл у якому введемо даний код:

                using System;
                class Hello  
                {
                    public static void Main()
                    {          
                        Console.WriteLine("Hello world!");
                    }
                }

Збережемо цей файл у форматі *.cs. Та запустимо на компілювання командою "csc.exe *.cs".
